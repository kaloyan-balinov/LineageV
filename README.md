A free, offline and open source family tree maker, designed for easy usability. Made with Godot!

**FEATURES:**
- Offline-first, intuitive way to visualise your own lineage
- Easy on the eyes!
- You can make "slides" with info about each member of your family tree
- A present mode built right into the program (eventually i would like to have a web visualiser)
- Snap-to-grid functionality
- Each member can have a custom colour asigned to them
- and much more
